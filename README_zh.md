# 仓颉API公共仓

## 简介

仓颉API公共仓，包含仓颉API接口声明，SDK构建等。

## 目录

```
├── api
│   └── xxxKit                   # Kit 具体名字，比如Cangjie, BasicServiceKit等
│       └── xxx.cj.d             # 各kit具体的接口文件，存放路径以kit实际文件列表为准
├── build-tools                  # sdk 构建工具
└── kits
    └── kit.xxx.cj.d             # 各kit对外统一声明文件
```
